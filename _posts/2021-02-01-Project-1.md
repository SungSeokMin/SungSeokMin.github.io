> ## Shape Of Voice

사람의 목소리 파동을 이용해 채도, 명도, 색을 다르게 나타내는 이미지를 생성하는 웹

### schema

- user : 각 유저들의 정보
- voice : 목소리에 대한 정보
- board : 게시글에 대한 정보
- voice_board : voice table과 board table의 join table
